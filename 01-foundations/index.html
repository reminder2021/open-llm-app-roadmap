Python 与 Web API 基础

LLM 应用本质上是 API、数据处理、服务编排和用户交互。这个阶段补齐工程基本功。

建议周期：1-2 周产出：一个 FastAPI 小服务

Python 必备能力

  • 虚拟环境、依赖管理、环境变量。
  • 文件读取、JSON/YAML、CSV、异常处理。
  • 类型标注、dataclass、Pydantic 数据模型。
  • 异步基础：async/await、超时、并发请求。
  • 日志：记录输入、输出、耗时和错误。

Web API 基础

  • HTTP 方法、状态码、Header、鉴权 Token。
  • REST API 与流式响应。
  • FastAPI 路由、请求体、响应模型。
  • OpenAI-compatible API 的请求格式。
  • 接口重试、限流、超时与错误处理。

实践项目

做一个“文本处理 API”：接收一段文本，返回摘要、关键词和结构化 JSON。先用规则实现，后续替换成 LLM 调用。

验收标准

  • 能启动本地服务。
  • 接口参数有校验。
  • 错误时返回清晰信息。
  • 有基础日志。
